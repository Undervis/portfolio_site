Музей обратился к нам с проблемой хранения записей об экспонатах

Они хранили все записи в PDF файлах распихивая их по папкам. Естественно ни о какой систематизации речи не идёт. Сортировка, фильтрация и поиск лишь в мечтах

Я разработал веб приложение которое позволяет хранить записи об экспонатах, редактировать их.
На главной странице можно просмотреть все записи, отсортировать их, отфильтровать по группам и фондам.

Так же, по техническому заданию, были разделены права пользователей на редакторов и директора

  • Дизайн приложения - Bootstrap модифицированный под брендбук музея

  • Основа - Django, Django MPTT

  • База Данных - MySQL

  • Скрипты - jQuery, Isotop, Bootstrap

На данный момент приложение уже год работает в музее

* Все скриншоты сделаны на билде работющем в самом музее
